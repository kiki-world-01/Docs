## 1. Get Your API Key

Every request to the Podium API requires an API key passed as a Bearer token. Keys are scoped to an organization and prefixed to indicate environment:

| Prefix | Environment | API Base URL |
|--------|------------|--------------|
| `podium_test_` | Staging | `https://api.test.podiumcommerce.xyz/api/v1` |
| `podium_live_` | Production | `https://api.podiumcommerce.xyz/api/v1` |

<Info>
  Contact the Podium team to provision an organization and receive your API keys. Each organization can have multiple keys with independent rate limits.
</Info>

## 2. Make Your First Call

Verify your key works by listing products:

<CodeGroup>

```bash cURL
curl -X GET https://api.podiumcommerce.xyz/api/v1/products \
  -H "Authorization: Bearer podium_live_your_key_here"
```

```typescript TypeScript SDK
import Podium from '@podiumcommerce/node-sdk';

const podium = new Podium({
  apiKey: 'podium_live_your_key_here'
});

const products = await podium.products.list();
console.log(products);
```

```typescript Fetch
const res = await fetch('https://api.podiumcommerce.xyz/api/v1/products', {
  headers: {
    'Authorization': 'Bearer podium_live_your_key_here'
  }
});
const products = await res.json();
```

</CodeGroup>

A successful response returns a JSON array of products scoped to your organization.

## 3. Install the TypeScript SDK

The SDK auto-detects staging vs production from your key prefix:

```bash
npm install @podiumcommerce/node-sdk
```

```typescript
import Podium from '@podiumcommerce/node-sdk';

const podium = new Podium({
  apiKey: process.env.PODIUM_API_KEY
});

// The SDK is organized by namespace
const product  = await podium.products.get('prod_abc123');
const creators = await podium.creators.list();
const user     = await podium.users.get('user_xyz');
```

<Tip>
  The SDK is published as `@podiumcommerce/node-sdk` on npm. It ships with full TypeScript types generated from the OpenAPI spec.
</Tip>

## Local Development

To run podium-elysia locally for development:

<Steps>
  <Step title="Prerequisites">
    Install [Bun](https://bun.sh) (>= 1.0) and Docker.
  </Step>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/kiki-world-01/podium-elysia.git
    cd podium-elysia
    bun install
    ```
  </Step>
  <Step title="Start infrastructure">
    ```bash
    bun run db:start    # PostgreSQL on port 5432
    bun run redis:start # Redis on port 6379
    ```
  </Step>
  <Step title="Configure environment">
    ```bash
    cp .env .env.local
    ```
    At minimum, set `DATABASE_URL`, `DIRECT_URL`, `REDIS_URL`, `PLATFORM_ORG_ID`, `API_KEY_PREFIX`, and `API_SHARED_KEY`.
  </Step>
  <Step title="Push schema and start">
    ```bash
    bun run db:push  # Sync Prisma schema + apply RLS policies
    bun run dev      # http://localhost:3030
    ```
    Verify at `http://localhost:3030/docs` for the interactive API explorer.
  </Step>
</Steps>
