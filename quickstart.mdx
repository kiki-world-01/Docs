## 1. Create Your Account

<Steps>
  <Step title="Sign up">
    Head to the [Podium Developer Portal](https://www.podiumcommerce.xyz/onboarding) and sign in with email, Google, or a crypto wallet (powered by Privy).
  </Step>
  <Step title="Create an organization">
    During onboarding you'll create an **Organization** — this is your tenant. All API keys, products, orders, and data are scoped to your organization.
  </Step>
  <Step title="Choose a plan">
    Pick a subscription tier that matches your usage. You can upgrade any time from the dashboard.

    | Tier | Price | Requests/Month | Rate Limit |
    |------|-------|---------------|------------|
    | **Builder** | Free | 10,000 | 5/sec |
    | **Growth** | $99/mo | 100,000 | 20/sec |
    | **Pro** | $499/mo | 1,000,000 | 50/sec |
    | **Enterprise** | Custom | Unlimited | Custom |
  </Step>
  <Step title="Generate an API key">
    The onboarding flow generates your first API key automatically. You can create additional keys from **Dashboard > Settings > API Keys**.

    Keys are prefixed by environment:

    | Prefix | Environment | API Base URL |
    |--------|------------|--------------|
    | `podium_test_` | Staging | `https://podium-staging.up.railway.app/api/v1` |
    | `podium_live_` | Production | `https://api.podiumcommerce.xyz/api/v1` |
  </Step>
</Steps>

<Warning>
  Your full API key is only shown once at creation time. Copy it immediately and store it securely. The dashboard only displays the key prefix after that.
</Warning>

## 2. Make Your First Call

Verify your key works by listing products:

<CodeGroup>

```bash cURL
curl -X GET https://api.podiumcommerce.xyz/api/v1/products \
  -H "Authorization: Bearer podium_live_your_key_here"
```

```typescript TypeScript SDK
import { createPodiumClient } from '@podiumcommerce/node-sdk'

const client = createPodiumClient({
  apiKey: 'podium_live_your_key_here'
})

const products = await client.product.list({ page: 1 })
console.log(products)
```

```typescript Fetch
const res = await fetch('https://api.podiumcommerce.xyz/api/v1/products', {
  headers: {
    'Authorization': 'Bearer podium_live_your_key_here'
  }
});
const products = await res.json();
```

</CodeGroup>

A successful response returns a JSON array of products scoped to your organization.

## 3. Install the TypeScript SDK

The SDK auto-detects staging vs production from your key prefix:

```bash
npm install @podiumcommerce/node-sdk
```

```typescript
import { createPodiumClient } from '@podiumcommerce/node-sdk'

const client = createPodiumClient({
  apiKey: process.env.PODIUM_API_KEY
})

// The SDK is organized by namespace
const products = await client.product.list({ page: 1 })
const creator  = await client.merchant.getBySlug({ slug: 'clean-beauty-co' })
const user     = await client.user.getUsernameByUsername({ username: 'jane' })
```

<Tip>
  The SDK is published as `@podiumcommerce/node-sdk` on npm. It ships with full TypeScript types generated from the OpenAPI spec.
</Tip>

## 6. Manage Your Keys

From the [Dashboard](https://www.podiumcommerce.xyz/dashboard/settings/api-keys) you can:

- **Create** additional API keys (up to your tier's limit)
- **Rotate** a key — generates a new key while the old one remains valid for 30 minutes
- **Enable/Disable** keys without deleting them
- **Delete** keys permanently

All key changes take effect immediately (cached keys expire within 15–150 seconds).
