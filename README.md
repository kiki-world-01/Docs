# Podium Documentation

Documentation for the [Podium](https://podium.build) commerce and agentic intelligence platform. Built with [Mintlify](https://mintlify.com).

## What's Documented

- **Platform** — Architecture, authentication, core concepts
- **Agentic Intelligence** — Enrichment pipeline, product feed, x402 payments, reference implementations
- **API Reference** — ~273 REST endpoints across commerce, engagement, and intelligence
- **Smart Contracts** — V1 Intent Settlers and V2 Task Pool system on Base
- **SDK & Guides** — TypeScript SDK, integration guides, and 55+ use cases

## Development

Install the Mintlify CLI:

```bash
npm i -g mint
```

Run locally:

```bash
mint dev
```

View at `http://localhost:3000`.
