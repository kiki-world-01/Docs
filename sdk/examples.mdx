## Product Catalog Management

### Create, Configure, and Publish a Product

```typescript
import Podium from "@podiumcommerce/node-sdk";
const podium = new Podium({ apiKey: process.env.PODIUM_API_KEY });

const product = await podium.products.create(creatorId, {
  name: "Organic Face Serum",
  slug: "organic-face-serum",
  description: "Lightweight hydrating serum with hyaluronic acid and vitamin C",
  price: 2500,
  maxDiscount: 500,
  media: [
    { url: "https://cdn.example.com/serum-hero.jpg", type: "IMAGE", order: 0 },
    { url: "https://cdn.example.com/serum-texture.jpg", type: "IMAGE", order: 1 },
  ],
});

await podium.products.update(creatorId, product.id, {
  supply: 200,
  pointEligibility: "ELIGIBLE",
  shippingType: "STANDARD",
  attributes: [
    {
      name: "Size",
      type: "VARIANT",
      variants: [
        { value: "30ml", price: 2500, supply: 100, order: 0 },
        { value: "50ml", price: 3500, supply: 60, order: 1 },
        { value: "100ml", price: 5000, supply: 40, order: 2 },
      ],
    },
  ],
});

await podium.creators.publishProduct(creatorId, product.id);
```

## Order and Checkout Workflows

### Stripe Checkout (Consumer App)

```typescript
const order = await podium.orders.create(userId, {
  productId: "clx9prod001",
  variantIds: ["clx9var_30ml"],
  quantity: 1,
});

await podium.points.applyToOrder(userId, order.id, { points: 500 });

const shippingQuote = await podium.orders.getShippingQuote(order.id, {
  firstName: "Jane",
  lastName: "Doe",
  address: {
    line1: "123 Main St",
    city: "San Francisco",
    state: "CA",
    postalCode: "94102",
    countryCode: "US",
  },
});

const { clientSecret } = await podium.orders.checkout(userId, order.id, {
  points: 500,
});
// Pass clientSecret to Stripe.js on the frontend

// After Stripe webhook confirms payment:
await podium.points.finalizeOrder(userId, order.id);
```

### x402 Checkout (Agent)

```typescript
const response = await fetch(
  `https://api.podiumcommerce.xyz/api/v1/x402/orders/${orderId}/pay`,
  { method: "POST" }
);

if (response.status === 402) {
  const requirements = response.headers.get("X-PAYMENT-REQUIREMENTS");
  const parsed = JSON.parse(atob(requirements));

  // Sign a USDC transfer matching the requirements
  const paymentPayload = await signUsdcTransfer({
    to: parsed.payTo,
    amount: parsed.maxAmountRequired,
    network: "base",
  });

  const receipt = await fetch(
    `https://api.podiumcommerce.xyz/api/v1/x402/orders/${orderId}/pay`,
    {
      method: "POST",
      headers: { "X-PAYMENT": btoa(JSON.stringify(paymentPayload)) },
    }
  );
}
```

## Campaign Workflows

```typescript

  title: "What moisturizer matches your skin?",
  startsAt: new Date(),

// Add products/options as swipeable items
// ...
```

## Companion Agent Workflows

### Full Agent Lifecycle

```typescript
// 1. Create user and intent profile
const user = await podium.users.create({
  email: "shopper@example.com",
  username: "skincare_fan",
});

const profile = await podium.companion.createProfile(user.id, {
  skinType: "Oily",
  concerns: ["Acne", "SPF protection"],
  priceRange: { min: 15, max: 50 },
  brands: ["CeraVe", "The Ordinary"],
  avoidances: ["fragrance", "alcohol"],
});

// 2. Populate catalog (one-time or periodic)
await podium.companion.batchCreateProducts({
  items: [
    {
      name: "CeraVe Moisturizing Cream",
      brand: "CeraVe",
      category: "moisturizer",
      price: 18.99,
      productUrl: "https://cerave.com/moisturizing-cream",
      imageUrl: "https://cdn.example.com/cerave.jpg",
    },
    {
      name: "The Ordinary Niacinamide 10%",
      brand: "The Ordinary",
      category: "serum",
      price: 5.9,
      productUrl: "https://theordinary.com/niacinamide",
      imageUrl: "https://cdn.example.com/niacinamide.jpg",
    },
  ],
});

// 3. Get AI-powered recommendations
const recs = await podium.companion.getRecommendations(user.id, {
  count: 5,
  category: "moisturizer",
});

// 4. Record user feedback
for (const product of recs.slice(0, 3)) {
  await podium.companion.recordInteraction({
    userId: user.id,
    productId: product.id,
    action: "RANK_UP",
  });
}

await podium.companion.recordInteraction({
  userId: user.id,
  productId: recs[3].id,
  action: "RANK_DOWN",
});

// 5. Place a concierge order
const order = await podium.companion.createOrder({
  userId: user.id,
  productId: recs[0].id,
  shippingAddress: {
    street: "123 Main St",
    city: "San Francisco",
    state: "CA",
    zip: "94102",
    country: "US",
  },
  email: "shopper@example.com",
});

// 6. Award points for engagement
await podium.companion.awardPoints(user.id, {
  amount: 25,
  details: { reason: "order_placed", orderId: order.id },
});

// 7. Track order
const orders = await podium.companion.listOrders(user.id);
const detail = await podium.companion.getOrder(order.id);
```

## Task Pool (On-Chain Bounties)

### Create and Fund a Task

```typescript
const task = await podium.tasks.create({
  amount: 5000000, // $5.00 USDC (6 decimals)
  duration: 86400, // 24 hours
  campaignId: "campaign_abc",
  verificationMethod: "Oracle",
  acceptanceCriteria: JSON.stringify({
    description: "Write a 200+ word product review",
    requirements: ["Personal experience", "Star rating 1-5", "Photo"],
  }),
});

const pools = await podium.tasks.getPools();
console.log("TaskPool:", pools.taskPool);
console.log("RewardPool:", pools.rewardPool);
```

### Solver Workflow

```typescript
// Register as solver
const solver = await podium.solver.register({
  walletAddress: "0xSolverAddress...",
});

// Browse available tasks
const tasks = await podium.solver.listTasks({ status: "OPEN" });

// Claim a task
await podium.solver.claim(taskId, { solverId: solver.id });

// Submit proof
await podium.solver.submit(taskId, {
  proof: "https://example.com/review-link",
  metadata: { wordCount: 342, rating: 4 },
});

// Check status — on-chain verification happens automatically
const status = await podium.tasks.getStatus(taskId);
// When verified, USDC is released from RewardPool to solver
```

```typescript
// Create a reward program
const reward = await podium.nfts.createReward({

// Redeem a reward

  network: "base",
  contractAddress: "0xNftContract...",
  tokenId: "42",
});
```

### Reward Types

| Type | Description |
|------|-------------|
| `FREE_PRODUCT` | Redeem for a free product |
| `DISCOUNT` | Percentage or fixed discount |
| `EVENT_PASS` | Access pass for events |
| `POINTS` | Bonus point grant |

## Agentic Commerce (Public Endpoints)

### Query the Agentic Product Feed

No API key required — designed for machine consumption:

```typescript
const response = await fetch(
  "https://api.podiumcommerce.xyz/api/v1/agentic/products/feed",
  {
    headers: {
      "X-INTENT-CONTEXT": JSON.stringify({
        concerns: ["hydration"],
        priceRange: { min: 10, max: 50 },
      }),
    },
  }
);

const products = await response.json();
// Each product includes intent scores and provenance metadata
```

## Error Handling

All SDK methods throw typed errors:

```typescript
import Podium, { PodiumError } from "@podiumcommerce/node-sdk";

try {
  const product = await podium.products.get(creatorId, "invalid_id");
} catch (error) {
  if (error instanceof PodiumError) {
    console.error(`${error.status}: ${error.message}`);
    // 404: Product not found
  }
}
```

Common error codes:

| Status | Meaning |
|--------|---------|
| `400` | Validation error (check request body) |
| `401` | Missing or invalid API key |
| `403` | Insufficient permissions for this resource |
| `404` | Resource not found |
| `409` | Conflict (e.g., duplicate slug) |
| `429` | Rate limit exceeded |
